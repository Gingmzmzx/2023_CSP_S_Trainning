一、矩阵

  • 简单理解：n行，m列的数组
  • 运算

    • 加/减
      对应位置相加/减即可

    • 乘/除
      对于A×B=CA \times B = CA×B=C，其中

      • AAA：nnn行mmm列
      • BBB：jjj行kkk列
      • CCC：nnn行kkk列

      必须满足条件

      • m=jm = jm=j
      • CCC的行数为AAA的行数，列数为BBB的列数

      对于CCC中的第aaa行第bbb列的数，取出AAA矩阵的第aaa行，BBB矩阵的第bbb列，对应位置相乘，然后相加。
